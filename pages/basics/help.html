What is

The whatis utility outputs a short description of another utility.

Syntax

whatis [utility]

Examples

whatis ls
whatis cat

Exercise

Use the what is utility to view a short description of the touch utility.

solution
whatis touch
 touch(1) - change file timestamps

Manual

man is a Unix utility that displays the manual for another utility. The manual provides the most information of any of the help features covered in this tutorial. It includes a description of the utility, syntax, options, exit statuses, and information on copyright, authorship and error reporting. The manual opens "on top of" the terminal rather than outputting text as the other help features do. Press q to exit the manual and return to the terminal.

Syntax

man [utility]

Examples

man ls
man cat

Exercise

Open the page of the Unix manual about the touch utility.

solution
man touch

Help Option

Many utilities have a help option. This option causes the utility to output the contents of its manual instead of performing its usual function.

Syntax

[utility] --help

Examples

ls --help
cat --help

Exercise

Print the manual's information on the touch utility to the terminal.

solution
touch --help

Apropos

The apropos utility outputs the names and descriptions of utilities that contain a given keyword in the headers of their manuals. It is useful for finding the right utility for a task when you do not know its name.

Syntax

apropos [keyword]

Examples

apropos list
apropos concatenate

Exercise

Use the apropos utility to display a list of Unix utilities that have to do with timestamps.

solution
apropos timestamp

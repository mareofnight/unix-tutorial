Print Working Directory

The working directory can be thought of as the directory that is currently open, or the directory you are "in". The print working directory utility outputs the name of this directory.

Syntax

pwd [options]

Examples

pwd

Exercise

Use the pwd utility to see what your working directory is right now. If you have not navigated to a different directory since starting the terminal, you will see the following output:

/home/your_user_name
solution
pwd

List

The list utility outputs a list of the files and directories in the working directory.

Syntax

ls [options]

Examples

ls
ls Documents
ls -l courses/409

Exercise

List the files and directories in your working directory. If your working directory contains other directories, list the contents of that directory as well.

solution
ls
ls [directory]

Change Directory

Syntax

cd [options] [directory]

Examples

cd courses/info_tech/409
cd data

Exercise

Change your working directory to one of the ones you saw in the previous exercise. Use the list utility again, and change your working directory to one from the list. Use print working directory to see where you are. Then, navigate back to the directory you started in.

Note: .. represents the parent of a directory. In computing, the parent of something is the thing that contains it. In the first part of this exercise, you moved two levels deeper into your computer's file structure. Using this command, you move outward two levels, back to where you started.

solution
cd [directory]
cd [directory]
../

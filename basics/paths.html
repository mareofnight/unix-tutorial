Absolute Paths

Absolute paths describe where a file is relative to the root directory or your home directory.

The root directory is the directory that contains all other files and directories. It is represented by a slash (/) at the beginning of a path (as opposed to the slashes between parts of a path).

A home directory is the main directory of a specific user. It is located at /home/[your username]. A tilde (~) can be used to represent your home directory.

Examples

/
/home/tim/courses/409
~
~/courses/409

Relative Paths

Relative paths describe where a file is relative to your working directory. A period (.) can be used to represent your working directory, but paths that do not start with a tilde or slash to indicate they are absolute are treated as relative to the working directory by default, so this is optional. A pair of periods (..) represent the parent of a directory; that is, the directory that immediately contains it. Pairs of periods can be strung together separated by slashes as though they are file names in order to refer to something more than one directory higher than the working directory.

Examples

Moving to a deeper directory
level4/level5
Moving up a level
..
Moving up two levels
../..
Moving up two levels and then down one
../../level2

Logging In

When logging in to a computer using a Unix based or Unix-like operating system that does not have a GUI, the terminal will prompt you for a user name and password. (In most distributions that have GUIs, this will be done through the GUI before opening the terminal.) Note that when typing your password, your input will either show up as a character such as * or •, or not be displayed at all.

Example

Username: user01
 Password: ********

Exercise

If you are using a version of Unix that requires logging in through the terminal, log yourself in. If you are using a Unix distribution that has a GUI, open the terminal application now.

Confirming Your Username

The whoami utility outputs your username. This is a method of checking which account you are logged in to.

Syntax

whoami [options]

Examples

whoami

Exercise

Confirm your username using the whoami utility. You can do this using the same command as the example above.

Logging Out

If using Unix without a GUI, you can log out using the logout

Examples

logout
exit

Exercise

Log out or exit the terminal, and then log back in or re-open the terminal.

Make Directory

mkdir creates a new directory at the specified path.

Syntax

mkdir [directory]

Examples

mkdir [options] documents/courses/409
mkdir temp

Exercise

Create a new directory named example within your current working directory. Then, create another directory named example2 within example without changing your working directory to do so.

Use the ls and cd utilities to check your work. Change your working directory back to the parent of example before beginning remove directory exercise.

solution
mkdir example
mkdir example/example2

Remove Directory

The rmdir utility deletes the directory at the specified path. Only empty directories can be removed.

Syntax

rmdir [options] [directory]

Examples

rmdir documents/courses/409
rmdir temp

Exercise

Try removing the example directory you created. Does it work? Why or why not? If not, find another way of using the rmdir utility to delete it. (This may entail using more than one command.)

solution
rmdir example

This does not work because non-empty directories cannot be removed.

rmdir example/example2
rmdir example

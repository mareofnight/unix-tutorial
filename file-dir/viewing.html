Concatonate

The concatonate utility outputs the contents of one or more files in succession. By default, the output is printed to the terminal, making the concatonate utility a fast and easy way to view the contents of small files.

Syntax

cat [options] [file name] [file name] [file name] [file name...]

Examples

cat usernames.txt
cat 2010_data 2011_data 2012_data

Exercise

Find a text file and use the concatonate utility to view it.

solution
cat [file name]

More and Less

The less utility opens a file in an interface that displays its contents a page at a time. There are several ways to navigate through the file, including the space bar (which advances to the next page) and the arrow keys. To exit, press q.

More is an older utility that also pages through large files. More is different from less in that it prints the file to the terminal rather than opening its own interface, and it only allows for paging forward, not backward. The more utility prints the next line when the user presses enter, or the next page when the user presses space.

Syntax

more [options] [file]
less [options] [file]

Examples

less story.txt
more story.txt

Exercise

Open your text file in both the more and less utilities. Note the differences in how they handle page turning.

solution
more [file name]
less [file name]

Head and Tail

Head and tail print the first or last few lines of a file, respectively, and print them to the terminal. The default number of lines is ten. Alternately, options can be used to specify a number of lines or bytes.

Syntax

head [options] [file]
tail [options] [file]

Output Length Option Syntax

When using single character options to specify the length of the output, the number of bytes or lines can be placed either before or after the character, but never before the dash.

head -c400 file
tail -5n file

When using options with a double dash and multiple characters, type a = after the option, and then the number of lines or bytes, with no spaces.

tail --bytes=150 file
head --lines=3 file

Examples

head names.txt
tail data

Exercise

Print the first and last lines of your file to the terminal.

solution
head --lines=1 [file name]
tail --lines=1 [file name]
